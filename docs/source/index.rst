Welcome to `gym-anm`!
===================================

**Train reinforcement learning agents to manage electricity networks.**

What is it?
-----------
`gym-anm` is a framework for designing reinforcement learning environments
(tasks) that model Active Network Management (ANM) tasks in electricity distribution networks.

What can I use it for?
----------------------

Key features
------------
#. it requires very little background in electricity systems modelling,
#. the resulting environments follow the OpenAI Gym structure, a well-established standard in the reinforcement learning community.
